Permit access to your blog only to the logged in users

== Description ==

This plugin enables you to permit access to your blog only to the logged in users.

== Installation ==
1. Upload `login_requiered.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
